# Customer Data Analysis Dashboard | Power BI

This project presents an interactive Power BI dashboard analyzing customer sales performance and business KPIs.

## Key Insights
✅ Designed and developed an end-to-end Power BI dashboard analyzing $25.3M revenue, 25.2K orders, and $10.5M profit (2020–2022)

✅ Tracked performance KPIs including Return % (1.09%) and monthly revenue growth (+3.31%)

✅ Identified Top 10 high-profit products ($54K+ profit each)

✅ Segmented 17K unique customers with $1.43K revenue per customer

✅ Analyzed category performance (17K+ orders in Accessories – highest segment)

✅ Performed geo-analysis across 6+ countries (North America, Europe, Pacific)

✅ Built Revenue, Orders & Profit vs Target analysis using dynamic KPI gauges

✅ Optimized data model using Star Schema (1 fact + 6 dimension tables)

✅ Implemented advanced DAX measures, slicers, bookmarks & multi-page navigation
